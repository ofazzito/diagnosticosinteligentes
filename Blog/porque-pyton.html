¿Por qué la Pyton?

Varios motivos que hacen hoy de Python un lenguaje estrella, es fácil de aprender, es un lenguaje de propósito general, es un lenguaje modular, implementa el paradigma de programación a objetos, hay una enorme comunidad, es software de código abierto, tiene una implementación amigable y completa de estructuras de datos, es un lenguaje muy expresivo.

Python ha sido ampliamente adoptado por la comunidad científica. Es el lenguaje preferido para Inteligencia Artificial: Keras y Tensorflow están hechos en Python.

Google también apuesta fuertemente a Python y se pueden ejecutar aplicaciones Python en un Drive de Google con Colaborative. Google Colab es una implementación Cloud de Jupyter Notebook, un popular IDE de Python.

Instagram, la mayor aplicación online para compartir fotos a nivel mundial utiliza Python en su backend, también lo hacen Spotify, Netflix, Uber, Dropbox, Pinterest, etc.
